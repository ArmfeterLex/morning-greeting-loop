# Morning Greeting Loop

Цикл на Python от 0 до 23, который выводит приветствие "Доброе утро!" для часов до 12.

## Функциональность

- Цикл от 0 до 23 включительно
- Проверка: если current_hour < 12
- Вывод строки "Сейчас X часов! Доброе утро!" для часов 0-11
- Для часов 12-23 ничего не выводится

## Пример работы

Сейчас 0 часов! Доброе утро!
Сейчас 1 часов! Доброе утро!
Сейчас 2 часов! Доброе утро!
...
Сейчас 10 часов! Доброе утро!
Сейчас 11 часов! Доброе утро!

## Куда можно встроить

- В программы с часами и приветствиями
- В учебные материалы по циклам
- Как пример фильтрации значений
- В демонстрацию работы range и условий

## Требования

- Python 3.x

## Примечание

Цикл перебирает все 24 часа, но выводит сообщение только для первой половины дня (0-11). Обратите внимание на склонение слова "часов" — для демонстрации используется фиксированная форма.
